# 🌸 Pink Login Page UI

A beautiful and responsive **Login Page UI** built using **HTML, CSS, and JavaScript**.  
This project features a soft pink aesthetic design with glassmorphism effects, modern input fields, and a stylish floral background.

---

# ✨ Features

- Modern Login Form Design
- Responsive Layout
- Glassmorphism UI Effect
- Stylish Pink Theme
- Remember Me Checkbox
- Forgot Password Option
- Register Redirect Text
- Beginner Friendly Project

# 🛠️ Technologies Used

- CSS3
- JavaScript
